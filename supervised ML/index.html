What is supervised machine learning?

This type of ML involves supervision, where machines are trained on labeled datasets and enabled to predict outputs based on the provided training. The labeled dataset specifies that some input and output parameters are already mapped. Hence, the machine is trained with the input and corresponding output. A device is made to predict the outcome using the test dataset in subsequent phases"

For example

consider an input dataset of parrot and crow images. Initially, the machine is trained to understand the pictures, including the parrot and crowâ€™s color, eyes, shape, and size. Post-training, an input picture of a parrot is provided, and the machine is expected to identify the object and predict the output. The trained machine checks for the various features of the object, such as color, eyes, shape, etc., in the input picture, to make a final prediction. This is the process of object identification in supervised machine learning.

Types of Suprevised learning

There are several types of supervised learning algorithms.

TYPE 1

Classification:

In classification tasks, the algorithm learns to classify inputs into predefined categories or classes. For example, given an image of an animal, the algorithm may classify it as a cat or a dog.

TYPE 2

Regression:

In regression tasks, the algorithm learns to predict a continuous numerical value based on the input data. For example, predicting the price of a house based on its features like size, location, and number of bedrooms.
